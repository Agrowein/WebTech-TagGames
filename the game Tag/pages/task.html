Задание 30. Игра "Пятнашки"

Скриншот 1. Внешний вид

Пятнашки – одна из известнейших миру головоломок. Она представляет набор, в который входит квадратная коробка, сторона которой равна 4 сторонам костяшки, то есть. 4х4. Внутри этой коробки 15 квадратных костяшек. В коробке остается 1 свободное место под одну костяшку. Цель игры – упорядочивание костяшек по порядку. Концом игры считается, когда все костяшки от 1 до 15 стоят друг за другом.

Логика веб-приложение реальзована на языке JavaScript.

Когда пользователь решит головоломку, появится окно с предложением записать результат в "таблицу лидеров" и полем для ввода имени.

После нажатия кнопки "ок" клиент сформирует фунцией fetch() POST запрос с данными об игроке и отправит его на сервер.

Сервер проверит поле с именем пользователя на корректность. В случае некорректного ввода сервер отправит ответ с кодом ошибки. В противном случае сервер обратиться к базе данных и сделает в ней запись с данными о пользователе (имя игрока, счет игры).

Далее сервер делает выборку базы данных, преобразует в JSON объекты и отправляет в качестве ответа клиенту.

Получив ответ, клиент на основе полученных данных формирует ячейки обновленной "таблицы лидеров".

Стек технологий
  • Страничка: HTML/CSS (использовался SCSS)
  • Логика головоломки: JavaScript
  • Back-end: PHP
  • База данных: MySQL
